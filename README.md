# Projeto de Estudos Web - Leonardo Borges

Este repositório faz parte da minha jornada de aprendizado em **Análise e Desenvolvimento de Sistemas (3º Semestre)** na Anhanguera Educacional.

## 🚀 Sobre o Projeto
Um espaço dedicado ao desenvolvimento de habilidades em tecnologias Front-end, focado em criar interfaces responsivas e organizadas.

## 🛠️ Tecnologias Utilizadas
* **HTML5**: Estruturação de conteúdo.
* **CSS3**: Estilização e responsividade (Mobile First).
* **Git & GitHub**: Controle de versão e portfólio profissional.
